Curso Básico de Python
Este curso básico de Python está diseñado para introducir a los estudiantes a los conceptos fundamentales de la programación en Python. A continuación se presenta una descripción general de los archivos y su contenido.

Estructura del Proyecto
condicion/
    condiciones.py
    condicionesWhile.py
datos compuestos/
    lista.py
    tupla.py
extras/
    cadenas.py
impresion/
    Concatenacion.py
    eje_1.py
input.py

Archivos y Descripción
condiciones.py
Este archivo contiene ejemplos de uso de condicionales en Python:

Condición simple if
Condición if-else
Condición if-elif-else
Condiciones anidadas
Condiciones múltiples con operadores lógicos
condicionesWhile.py
Este archivo muestra el uso de condicionales dentro de un bucle while:

Uso de if, if-else, y if-elif-else dentro de un bucle while
[datos compuestos/lista.py](datos compuestos/lista.py)
Este archivo presenta ejemplos de manipulación de listas en Python:

Acceso a elementos de la lista
Modificación de elementos
Métodos de lista como append y remove
Uso de la función len
[datos compuestos/tupla.py](datos compuestos/tupla.py)
Este archivo contiene ejemplos de uso de tuplas en Python:

Acceso a elementos de la tupla
Conversión entre tuplas y listas
Concatenación de tuplas
Desempaquetado de tuplas
cadenas.py
Este archivo muestra diversas operaciones con cadenas de texto en Python:

Concatenación de cadenas
Repetición de cadenas
Acceso a caracteres y subcadenas
Métodos de cadena como upper, lower, replace, split, join, y strip
Uso de operadores in para verificar la existencia de subcadenas
Formateo de cadenas con f-strings
Concatenacion.py
Este archivo contiene un ejemplo simple de concatenación de cadenas e impresión del resultado.

eje_1.py
Este archivo presenta un script básico de Python que incluye:

Impresión de "Hello, World!"
Operaciones aritméticas básicas: suma, resta, multiplicación, división, módulo, exponenciación, y división entera
input.py
Este archivo muestra cómo solicitar al usuario que ingrese datos y cómo imprimir los datos ingresados.
